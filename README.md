# Ignore the Blueprint

Personal website of GUAN Xiaoyu (a.k.a. ZHANG Damao).

This site is:

* Built with [Pelican][0]
* Managed with [pyenv][1] and [pip][2]
* Deployed on [CloudFlare][3]

The content is 100% hand written. Some of the metadata is [generated by GPT][4].

## Setup

Set up the local environment with `pyenv`, `venv`, and `pip`:

```bash
pyenv install 3.11.3
pyenv local 3.11.3
python -m venv .env
source .env/bin/activate
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
ollama pull qwen3.5:0.8b
```

Make sure Ollama is installed separately and that the model you want to use is already pulled locally.

## Common Commands

Build the site:

```bash
make html
```

Serve the site locally with auto reload:

```bash
make devserver PORT=8000
```

## Generate post headers

Post headers are generated locally with Ollama.

Run `python generate_post_header.py` to scan untracked markdown files under `content/posts/`.

Run `python generate_post_header.py content/posts/my-post.md` to process specific files.

Run `python generate_post_header.py --model llama3.2 content/posts/my-post.md` to override the default model.

Run `python generate_post_header.py --think content/posts/my-post.md` to enable model thinking output. Thinking is disabled by default.

Run `python generate_post_header.py --timeout 30 content/posts/my-post.md` to fail faster when the local Ollama request stalls. Use `--timeout 0` to wait indefinitely.
